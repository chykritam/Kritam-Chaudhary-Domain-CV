Bootstrap Framework

Responsive Grids

I found Bootstrap incredibly helpful for saving time. Instead of writing custom CSS for every layout, I used Bootstrap's built-in grid classes.

In the farm dashboard example on the left, I used classes like col-md-6 to make the boxes sit next to each other on a laptop, but stack neatly on top of each other if you view this on a phone.

This is called responsive design—the layout adapts to any screen size automatically.
